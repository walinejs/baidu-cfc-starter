# [Waline](https://github.com/walinejs/baidu-cfc-starter)


Waline 是一款基于 Valine 衍生的简洁、安全的评论系统，该项目是使用[百度函数计算](https://console.bce.baidu.com/cfc/#/cfc/functions)部署的一个 [Waline](https://waline.js.org) 应用示例。
